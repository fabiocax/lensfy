Bem-vindo ao Lensfy

O Lensfy roda localmente no seu computador. Para proteger o acesso às APIs e à interface, vamos gerar um token de dispositivo exclusivo desta máquina — sem login nem senha.

  • Acesso restrito a este equipamento (apenas loopback).
  • O token fica salvo com segurança em ~/.lensfy/device_token.
  • Gerado uma única vez; não precisa decorar nada.

Guarde-o apenas se for acessar a API por ferramentas externas (header X-Lensfy-Token). Ele permanece neste equipamento.
